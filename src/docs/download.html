Instruction to download ViroBLAST

We provide the stand-alone ViroBLAST package for the maximum utility in other research fields. It has been tested in Linux machine. Users can download the package by clicking the download button below, install it in their own machine, configure and customize the sequence databases they need. Users can then manage their own ViroBLAST web server tailored to meet their specific research purpose.

After downloading ViroBLAST package, place it into document directory of your HTTPD server and uncompress it by

gzip -d viroblast.tar.gz
tar -xvpf viroblast.tar
The paramenter "p" in tar options is important. It will preserve file access options stored in the distrubution.
